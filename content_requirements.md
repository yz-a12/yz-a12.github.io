# AcadHomepage 信息收集模板

请按照下表逐项填写你希望展示在个人主页上的全部内容。所有内容都支持 Markdown/HTML，可以直接在表格中备注格式需求或额外说明。

| 模块 | 需要提供的具体内容 | 你的填写 |
| --- | --- | --- |
| 站点配置 | `title`、`description`、`repository`、`google_analytics_id`（可选）、SEO 验证 token（可选） |  |
| 作者信息 | 姓名、头衔、单位、所在城市/国家、邮箱、社交链接（GitHub/LinkedIn/ORCID/Google Scholar 等） |  |
| 头像与 favicon | 头像图片文件路径或 URL、favicon 压缩包或生成方式 |  |
| 研究方向 | 3-5 个关键词或短句，支持中英文混排 |  |
| 教育经历 | 每条包含学位、学校、时间段、地点，可附荣誉或导师信息 |  |
| 工作/任职 | 每条包含职位、机构、时间段、职责概述 |  |
| 科研成果 | 论文/出版物列表：标题、发表渠道、年份、链接、是否显示引用数（如需提供 Google Scholar paper ID） |  |
| 项目/软件 | 项目名称、角色、技术栈、主要贡献、访问链接（GitHub/Demo/视频等） |  |
| 荣誉与奖项 | 奖项名称、授予机构、年份、简短介绍 |  |
| 教学/服务 | 授课课程、助教经历、会议服务、程序委员会等信息，附时间和职责 |  |
| 媒体/讲座 | 新闻报道、受邀报告、播客或访谈标题、链接、日期 |  |
| 下载资源 | 简历、Research Statement、Teaching Statement 等文件路径/URL，以及希望的展示文字 |  |
| 其他页面需求 | 需要新增的 `_pages/xxx.md` 页面及内容大纲，例如 Projects、Blog、News 等 |  |
| 语言与展示策略 | 是否需要中英文双语、分栏布局、折叠块或特殊交互需求 |  |

## 使用说明
1. 将上述表格复制到你喜欢的编辑器中并填写第三列 “你的填写”。
2. 如果某项暂时不需要，可以写 `无` 或留空。
3. 回传填写结果后，我会根据内容更新 `_config.yml`、`_pages/about.md` 等文件，并在需要时创建新页面。

> **提示**：若有自定义样式需求（颜色、字体、模块顺序），可以在表格下方单独写出补充说明。
